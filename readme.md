# Скрипт для составления графика рабочих смен.

## Автор проекта Никандров Владислав  
`Инструменты и стек: #python #pandas`
[Репозиторий на GitHub](https://github.com/Doshaniu/Work-shift-schedules)

## Данный проект представляет из себя скрипт который парсит данные о количестве сотрудников, графике их отпусков, праздничных днях, и формирует exel файл с графиком рабочих дней на месяц.

## Как развернуть проект локально
>1. [Клонируйте репозиторий]((https://github.com/Doshaniu/Work-shift-schedules))
>2. Создайте и активируйте виртуальное окружение командами
>
> ```python3 -m venv venv```  
> ```source venv/scripts/activate```
>3. Установите зависимости из файла requirements.txt
>
> ```python.exe -m pip install --upgrade pip```  
> ```pip install -r requirements.txt```  
> 
>4. В директорию input data положите exel файл со списком сотрудников и их графиком отпусков.
>5. Откройте файл to_exel.py и в строке MONTH введите число обозначающее нужный месяц.
>6. Запустите файл to_exel.py с помощью терминала командой python .\to_exel.py
>7. Перейдите в директорию output_data и откройте создавшийся файл.
>8. Для того чтобы создать график для других месяцев измените значение константы MONTH на нужный месяц.
